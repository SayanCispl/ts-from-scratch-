# TypeScript Playground 🚀

A personal repository to learn, practice, and experiment with **TypeScript**.  
This repo contains examples, exercises, and mini-projects that demonstrate TypeScript fundamentals and advanced features.

---

## 📚 Goals
- Understand TypeScript basics (types, interfaces, enums, generics).
- Explore advanced concepts (decorators, type inference, utility types).
- Practice integrating TypeScript with Node.js and frameworks like React.
- Build confidence by solving small coding challenges in TypeScript.

---

## 🛠️ Setup
1. Install [Node.js](https://nodejs.org/) (comes with npm).
2. Install TypeScript globally:
   ```bash
   npm install -g typescript
